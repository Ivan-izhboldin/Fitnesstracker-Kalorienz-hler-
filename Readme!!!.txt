Programmbeschreibung:

Das Programm soll den Benutzer nach seinem Geschlecht fragen und anschließend je nach Geschlecht die tägliche Kalorienzufuhr festlegen. Das Programm soll den Benutzer nach den Namen der über den Tag verzehrten Lebensmittel sowie deren Kaloriengehalt fragen. Das Programm soll die vom Benutzer eingegebenen Daten in einer Textdatei speichern und sie dem Benutzer auch auf dem Bildschirm anzeigen. Das Programm soll berechnen, ob der Benutzer seine tägliche Kaloriennorm eingehalten oder überschritten hat. Außerdem soll das Programm den Benutzer mit einer Bildschirmnachricht motivieren, mehr Sport zu treiben, falls er seine tägliche Kaloriennorm überschritten hat. Am Ende soll sich das Programm beenden und die Konsole leeren.
